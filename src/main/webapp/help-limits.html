Optional simulations limits. When limits are exceeded build will fail. Leaving or setting value to -1 will disable given limit. Exception is simulation name that needs to be filled

Simulation name field describes what gatling simulation given limits are linked to.
Max failed descibes how many failed request are acceptable before failing build
Mean number of responses per second describes how many responses need to come on average per second
Max response time describes how long longest reqest can take (time from request to response)
Mean response time describes how long on average should call take
